# Yet Another EfficientDet Pytorch

## Training

```python
python train.py -c 1 -p adult-kid --batch_size 11 --lr 1e-5 --num_workers 13 --num_epochs 100 --load_weights weights/efficientdet-d0.pth --train_machine_name foundation-pre
```

## Evaluation

```python
python coco_eval.py -p adult-kid -w logs/adult-kid-v3.1-base-coco-format/efficientdet-d1_99_48000.pth -c 1
```
